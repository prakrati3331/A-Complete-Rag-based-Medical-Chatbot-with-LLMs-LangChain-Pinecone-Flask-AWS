# A-Complete-Rag-based-Medical-Chatbot-with-LLMs-LangChain-Pinecone-Flask-AWS

 # How to run?
 ###  STEPS:

 Clone the repository

 ```bash
 git clone https://github.com/prakrati3331/A-Complete-Rag-based-Medical-Chatbot-with-LLMs-LangChain-Pinecone-Flask-AWS.git
```

### Step 01- Create a conda environment after opening the repository 

```bash
conda create -n ragbot python=3.10 -y
```

```bash
conda activate ragbot
```


### STEP 02- install the requirements

```bash
 pip install -r requirements.txt
  ```
